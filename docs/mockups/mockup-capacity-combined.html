Storage utilization %
Actual Storage Utilization = Total Data Stored (GB) / Total Storage available per cluster (GB) = 0.93 / 18.21 = 5.12%
Data Stored (GB) = Master object count × Replication factor × Average record size (bytes) / 1024³ = 0.93
Total Available Storage (GB) = Nodes × Devices per node × Device size (GB) × (1 − 2.4%) = 18.21
